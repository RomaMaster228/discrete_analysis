\CWHeader{Лабораторная работа \textnumero 7}

\CWProblem{Задан прямоугольник с высотой $n$ и шириной $m$, состоящий из нулей и единиц. Найдите в нём прямоугольник наибольшой площади, состоящий из одних нулей.

{\bfseries Формат входных данных:} В первой строке заданы \textbf{0 < n, m < 501}. В последующих $n$ строках записаны по $m$ символов 0 или 1 - элементы прямоугольника.

{\bfseries Формат результата:} { Необходимо вывести одно число – максимальную площадь прямоугольника из одних нулей. }

}
