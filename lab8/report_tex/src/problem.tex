\CWHeader{Лабораторная работа \textnumero 8}

Разработать жадный алгоритм решения задачи, определяемой своим вариантом. Доказать его корректность, оценить скорость и объём затрачиваемой оперативной памяти.

Реализовать программу на языке С или С++, соответсвующую построенному алгоритму.

\CWProblem{
Дана последовательность длины \textbf{N} из целых чисел \textbf{1, 2, 3}. Необходимо найти минимальное количество обменов элементов последовательности, в результате которых последовательность стала бы отсортированной.

{\bfseries Формат входных данных:} Число \textbf{N} на первой строке и \textbf{N} чисел на второй строке.

{\bfseries Формат результата:} { Минимальное количество обменов. }

}
